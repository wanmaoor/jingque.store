## 1.2 布局

### 标题目录

```
标题与目录：
# 标题1
## 标题2
### 标题3
#### 标题4
##### 标题5
###### 标题6
[toc] 自动生成目录

无序列表：
使用 * + - 都可以表示无序列表，还可以使用table键来创建多级列表。

有序列表：
直接使用数字伙子字母序号
```

### 表格

```
Markdown绘制表格：
|名称|效果|代码|

HTML绘制表格：
<table><tbody>
    <tr>
        <th rowspan="3">三行</th>
        <th><font color="Hotpink">列</font></th>
    </tr>
    <tr>
        <td bgcolor="Hotpink">单元格</td>
    </tr>
    <tr>
        <td>单元格</td>
    </tr>
</table>
```
